CodeBook

CodeBook is a Django webapp built for the final project of 15-437 at Carnegie Mellon University. It is an all-inclusive, go-to source for anyone looking to learn a new programming language.

CodeBook is a Web Application that suggests example open-source projects (found on GitHub), based on a user’s preferred language and level of experience. Users can rate projects based on how well documented they are, and tag projects with the specific technologies being used. The goal is not to provide a specific tutorial but rather to allow beginners to see what code looks like in real life.

  • Source: https://github.com/nshoemaker/CodeBook
  • Date: December 2014
  • Technologies: Django, Python, JavaScript, HTML, CSS, jQuery, Ajax, AWS, MySQL, GitHub API.

Glob3 Mobile Open Source Contribution

As a part of Facebook's Open Academy program, I contributed to an open source project called Glob3 Mobile.

Open Academy provides practical software engineering experiences through collaboration with top universities and open source projects around the world.

The program is comprised of two main components. It starts with the kickoff event in which all parties come together at Facebook Headquarters for an intensive weekend of learning, hacking and building connections. After the kickoff session, students return to their home universities and continue work in virtual teams.

I worked on Glob3 Mobile, a multiplatform 3D globe visualization framework.

I created an Android app that uses the Glob3 Mobile framework to plot the trajectory of the International Space Station in real time (code can be found here).

With Pratik Prakash, I worked on a feature that uses data from OpenStreetMap Buildings to render 3D buildings on a tiled basis.

  • Source: Start Bootstrap
  • Date: Spring 2015
  • Technologies: C++, C, Java, Android, iOS, OpenStreetMap, OpenGL

Wrist Ranger

The Wrist Ranger is a haptic feedback device for the visually impaired. Through a Small Undergraduate Research Grant (SURG) from Carnegie Mellon University, we looked into finding an effective way to provide a spacial representation of the surrounding environment for the visually impaired.

Won a Yahoo! Undergraduate Research Award at the 2014 Carnegie Mellon Meeting of the Minds Research Symposium.

This project looked to find an effective way to provide the visually impaired with spatial feedback of their surroundings by expanding upon past efforts of other researchers, as well as addressing issues that these other efforts faced, including problems with responsiveness, portability, and ease of use.

The product consists of multiple wrist-mounted ultrasonic range finders. The data collected about the surroundings is translated into different types of haptic or audible feedback, and each type of feedback will be tested for effectiveness.

Project mentioned in The Tartan, Carnegie Mellon's student-run newspaper: https://thetartan.org/2014/2/24/scitech/techbridge.

  • Date: May 2014
  • Technologies: Arduino
